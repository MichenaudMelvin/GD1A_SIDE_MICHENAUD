# **English**

## **How to launch the game ?**
Download zip, execute moongoose and put "http://localhost:8000/" in your navigator

or go to https://michenaudmelvin.github.io/GD1A_SIDE_MICHENAUD/ (recommended)

## **Gameplay :**
You can move on the x axis with directional arrows :
-----------------
<img src="forReadMeOrPresentation/captures/deplacement.gif" width="150" height="81">

When you kill an astronaut, you collect his jetpack :

And obviously with this jetpack, you can fly :

-----------------

## **About**
This is a small game made with [phaser 3](https://github.com/photonstorm/phaser)  
I made assets with [Adobe Photoshop](https://www.adobe.com/products/photoshop.html) and also used [tiled](https://github.com/mapeditor/tiled) to create the architecture of the level
